# ¿Cómo correr el código?

Para ejecutar el script de *FMIndex.java*

Se debe ejecutar el siguiente comando.

## Windows

`java -Xmx4g -cp lib\NGSEPcore_3.2.0.jar;bin uniandes.algorithms.fmindex.EntryPoint`

## Mac

`java -Xmx4g -cp lib/NGSEPcore_3.2.0.jar;bin uniandes.algorithms.fmindex.EntryPoint`
